Temas del Curso de Programacion Web

Unidad 1: HTML5

Estructura basica de un documento HTML
Etiquetas de texto: h1-h6, p, span, strong, em
Listas: ol, ul, li
Tablas: table, tr, td, th
Formularios: form, input, button, select, textarea
Multimedia: img, video, audio
Estructura semantica: header, nav, main, section, article, footer

Ejemplo de estructura HTML5:

<!DOCTYPE html> <html lang="es"> <head> <meta charset="UTF-8"> <title>Mi Pagina</title> </head> <body> <header>...</header> <main>...</main> <footer>...</footer> </body> </html>

Unidad 2: CSS3

Selectores: clase, id, elemento, descendente
Box model: margin, padding, border
Flexbox y Grid layout
Colores, fondos y gradientes
Tipografia: font-family, font-size, font-weight
Animaciones y transiciones
Responsive design con media queries

Ejemplo de CSS3:

.contenedor { display: flex; gap: 20px; flex-wrap: wrap; } @media (max-width: 600px) { .contenedor { flex-direction: column; } }

Unidad 3: Enlaces

Enlace relativo: href="pagina.html"
Enlace absoluto: href="https://www.ejemplo.com"
Atributo target="_blank" para abrir en nueva pestana
Enlace ancla: href="#seccion"
Enlace a correo: href="mailto:correo@ejemplo.com"
Enlace de descarga: atributo download

Unidad 4: Formularios

Tipos de input: text, email, password, number, date
Atributos: required, placeholder, name, id, value
Elementos select y option
Textarea para texto largo
Botones: submit, reset, button
Validacion HTML5 nativa
